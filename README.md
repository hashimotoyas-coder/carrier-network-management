# キャリアネットワーク開通管理システム

海外キャリアとの通信ネットワーク開通業務を管理するKintone風アプリケーション

## 📋 概要

14ステップのワークフローで、営業活動から技術設定、開通試験まで一元管理します。

### 主な機能

- **マスターファイル管理**: キャリア基本情報の登録・編集
- **進捗管理表**: 案件の進捗状況をリアルタイム追跡
- **オーダー依頼表**: 設備設定を自動生成・管理
- **ダッシュボード**: 全体状況を一目で把握

## 🚀 使い方

### ローカルで実行

1. このリポジトリをクローン
```bash
git clone <repository-url>
cd ローミング案件管理
```

2. ブラウザで開く
```bash
open index.html
```

### オンラインで使用

GitHub Pagesで公開中: `https://<username>.github.io/<repository-name>/`

## 💾 データ保存

データはブラウザのlocalStorageに保存されます。
- 同じブラウザであれば再訪問時もデータが保持されます
- ブラウザのキャッシュをクリアするとデータが消えます
- 重要なデータは定期的にエクスポートすることを推奨

## 🛠️ 技術スタック

- HTML5
- CSS3 (Kintone風デザイン)
- Vanilla JavaScript (フレームワーク不使用)
- localStorage (データ永続化)
